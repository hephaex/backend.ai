## üîç Monitoring

Check the status of all components:
```bash

kubectl get services -l app.kubernetes.io/instance={{ .Release.Name }}
```

View logs for specific components:
```bash

kubectl logs -l app.kubernetes.io/name=backend-ai-web-server -f
```
